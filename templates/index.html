About BloodLens

Built for Clinical Impact

The Problem

Manual blood cell analysis requires expert hematologists spending hours examining slides under microscopes — a process prone to fatigue-induced error, limited scalability, and high cost. In resource-constrained settings, this leads to delayed or missed diagnoses.

Our Solution

Blood Lens automates blood cell classification and cancer detection using convolutional neural networks trained on thousands of annotated microscopy images, delivering clinical-grade accuracy in under 2 seconds per sample.

System Architecture

01

Image Input & Preprocessing

Microscopy image received via Flask API and preprocessed (resizing and scaling) before model inference

→
02

Cell Classification (CNN)

Custom CNN trained on Blood Cell dataset (~17k images). Classifies input into 8 cell types with high accuracy

→
03

Cancer Detection (MobileNetV2)

Transfer learning using MobileNetV2 on cancer detection dataset (~3k images). Predicts 4 cancer stages.

→
04

Prediction Output

Both models generate predictions simultaneously. Final output includes cell type, cancer stage, and confidence scores returned via API
